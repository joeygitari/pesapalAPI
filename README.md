# PesaPal API Integration

### By Joanne Gitari 

## Description
This API allows you to get the Pesapal iframe URL by consuming the Pesapal API v3.0. It authenticates using JWT Tokens.

## Setup Requirements

* Git
* Github
* PHP
* MySql
* PesaPal API v3.0
* Postman
